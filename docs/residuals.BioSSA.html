residuals.BioSSA {BioSSA}R Documentation

Residuals of BioSSA decomposition

Description

Returns BioSSA residuals in the specified model.

Usage

## S3 method for class 'BioSSA2d'
residuals(object, groups, model = "additive", offset = 0, ...)
## S3 method for class 'BioSSA3d'
residuals(object, groups, model = "additive", offset = 0, ...)
## S3 method for class 'BioSSA2d.reconstruction'
residuals(object, model = "additive", offset = 0, ...)
## S3 method for class 'BioSSA3d.reconstruction'
residuals(object, model = "additive", offset = 0, ...)

Arguments

object

object of class ‘BioSSA2d’, ‘BioSSA3d’, ‘BioSSA2d.reconstruction’ or ‘BioSSA3d.reconstruction’

groups

numeric vector, indices of elementary components

model

model name ('additive', 'multiplicative' or 'poisson') or multiplicity power value

offset

numeric value, trend offset

...

additional arguments, passed to reconstruct

Value

Object of class ‘embryo2d’ or ‘embryo3d’ for 2d or 3d method correspondingly.
