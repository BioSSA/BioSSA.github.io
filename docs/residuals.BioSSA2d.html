residuals.BioSSA2d {BioSSA}R Documentation

Residuals of BioSSA2d decomposition

Description

Функция возвражает остатки. Опциоонально можно передать модель и сдвиг, тогда еще и поделит.

Usage

## S3 method for class 'BioSSA2d'
residuals(object, groups, ..., model = "additive", offset = 0)

Arguments

object

object of class ‘BioSSA2d’

groups

numeric vector, indices of elementary components

...

additional arguments, passed to reconstruct

model

model name ('additive', 'multiplicative' or 'poisson') or multiplicity power value

offset

numeric value, trend offset

Value

Object of class ‘embryo2d’.
