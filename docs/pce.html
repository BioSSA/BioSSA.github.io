pce {BioSSA}R Documentation

Read gene data from PCE file

Description

Read gene expression data from PointCloud File

Usage

read.pce(file)
read.pce.gene(file, gene, ...)
extract.gene.pce(pce, gene, measurement = c("default", "apical", "basal", "nuclear", "cellular"))

Arguments

file

character, PCE file name

gene

character, gene name for extraction

...

further arguments passed to extract.gene.pce (e.g. measurement)

measurement

measurement kind

pce

parsed PCE content returned by read.pce

Details

read.pce.gene is combination of read.pce and extract.gene.pce.

Value

Object of class ‘embryo3d’ for read.pce.gene and extract.gene.pce. Data frame with additional attributes for read.pce.

References

PCE format description: http://bdtnp.lbl.gov/Fly-Net/bioimaging.jsp?w=pceFormat

See Also

embryo3d, plot.embryo3d

Examples

file <- system.file("extdata/pce", "v5_s12166-23au07-22.pce", package = "BioSSA")
pce <- read.pce.gene(file, gene = "eve")
plot(pce, type = "hull")

## 
##      PLEASE NOTE:  As of version 0.3-5, no degenerate (zero area) 
##      regions are returned with the "Qt" option since the R 
##      code removes them from the triangulation. 
##      See help("delaunayn").

plot(pce, type = "nuclei")
