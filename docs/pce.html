pce {BioSSA}R Documentation

Read gene data from PCE file

Description

Read gene expression data from PointCloud File Format

Usage

read.pce(file)
read.pce.gene(file, gene, ...)
extract.gene.pce(pce, gene, measurement = c("default", "apical", "basal", "nuclear", "cellular"))

Arguments

file

charater, PCE file name

gene

character, gene name for extraction

...

further arguments passed to extract.gene.pce (e.g. measurement)

measurement

measurement kind

pce

data frame, parsed PCE content

Details

Some details?..

Value

Object of class ‘embryo3d’ for read.pce.gene and extract.gene.pce. Data frame for read.pce.

References

Format description: http://bdtnp.lbl.gov/Fly-Net/bioimaging.jsp?w=pceFormat

See Also

embryo3d, plot.embryo3d

Examples

file <- system.file("extdata/pce", "v5_s12166-23au07-22.pce", package = "BioSSA")
pce <- read.pce.gene(file, gene = "eve")
plot(pce, type = "hull")
plot(pce, type = "nuclei")
